\begin{abstract}
  We introduce a basic geometric type theory. This is meant to be used for
  writing down database schemata and their integrity constraints.
\end{abstract}

This type theory is inspired by the work of \textcite{vickers_1992}. The
particular presentation is closest to \textcite{kaposi_2019,kovacs_2022}.
